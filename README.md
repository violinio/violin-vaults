# Violin Vaults
The Violin Vaults provide a safety-centric defi vault infrastructure. It is provided as-is.

## Deploy
```
yarn deploy avax    
```

### Environment variables
- PRIVATE_KEY
- ETHERSCAN_APIKEY

## Contracts
The contracts have been deployed as-is on a variety of chains.
